# 0.6: New note in Single page app diagram
```mermaid
sequenceDiagram
    participant browser
    participant server
    
    Note right of browser: User writes a note in the text field and clicks Save
    
    Note right of browser: Browser executes event handler that adds note to list and re-renders the page
    
    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
    activate server
    Note right of browser: Browser sends the new note as JSON data
    Note left of server: Server saves the note and responds with status message
    server-->>browser: {"message": "note created"}
    deactivate server
    
    Note right of browser: The note is already visible - no page reload needed
```
